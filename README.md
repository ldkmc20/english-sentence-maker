# 영어 문장 만들기

갤럭시 S25 플러스 같은 안드로이드 휴대폰에서 한국어 음성 인식을 켜고, 하루 동안 자주 말한 문장을 영어 회화 문장으로 정리하는 PWA 첫 버전입니다.

## 가능 여부

- 가능: 사용자가 직접 `녹음 시작`을 누른 학습 세션 동안 한국어 음성을 인식하고, 문장을 브라우저 안에 저장하고, 많이 쓴 문장을 영어로 정리할 수 있습니다.
- 제한: 안드로이드와 브라우저 정책상 화면이 꺼진 상태에서 하루 종일 조용히 백그라운드로 듣는 방식은 안정적이지 않고 권장되지 않습니다.
- 더 강한 버전: 네이티브 Android 앱으로 만들면 포그라운드 서비스, 알림, 로컬 DB, 배터리 예외 설정을 붙여 더 길게 동작시킬 수 있습니다.

## 실행

배포된 앱 주소:

https://ldkmc20.github.io/english-sentence-maker/

```powershell
npm start
```

PC에서는 `http://localhost:4173`으로 확인할 수 있습니다.

휴대폰에서 음성 인식을 제대로 쓰려면 HTTPS 주소가 필요합니다. 정적 사이트로 배포한 뒤 갤럭시 Chrome 또는 Samsung Internet에서 열고 홈 화면에 추가하는 방식이 가장 간단합니다.

## 사용 방법

1. 앱을 HTTPS 주소로 엽니다.
2. `녹음 시작`을 누르고 마이크 권한을 허용합니다.
3. 한국어로 말하면 문장이 저장됩니다.
4. `많이 쓴 문장`에서 반복 문장을 확인합니다.
5. OpenAI API 키를 입력하고 `영어로 정리`를 누르거나, `프롬프트 복사`로 다른 AI 도구에 붙여 넣습니다.

## 개인정보

- 음성 녹음 파일은 저장하지 않습니다.
- 인식된 텍스트는 현재 브라우저의 `localStorage`에 저장됩니다.
- OpenAI API 키를 입력하면 브라우저에서 OpenAI API로 직접 요청합니다. API 키는 앱 저장소에 남기지 않습니다.
- 민감한 대화는 기록하지 않는 것이 좋습니다.
